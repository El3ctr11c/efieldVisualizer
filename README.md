# Electric Field Visualizer
This project visualizes the electric field generated by point charges using `matplotlib` and `numpy`. The main program reads a text file containing charge information and displays a 2D electric field plot.

## Getting Started

### Prerequisites

Make sure you have Python installed (3.7 or higher recommended) and the following packages:

```bash
pip install matplotlib numpy
```

## Usage (Running main.py)
### 1). Prepare a text file containing your charge data. Each line in the file must contain exactly three numbers, separated by spaces:

```
<charge_value> <x_position> <y_position>
```
#### charge_value: The value of the point charge (positive or negative).
#### x_position: The x-coordinate of the charge.
#### y_position: The y-coordinate of the charge.


### 2). (MAKE SURE TEXT FILE IS IN THE TESTS FOLDER)
### 3). Run main.py
```
python main.py
```
